#### 1. 安装依赖包


```
npm install
```


#### 2. 开发模式运行
提示：如果本地没有部署后端服务，可以使用演示版本的API接口，修改src/setting.js文件，将serverUrl改为'http://120.92.142.115:8080'
```
npm run serve
```

#### 3. 打包用于生产环境
```
npm run build
```

#### 4. 浏览器兼容性
IE 11，Chrome（及同内核的浏览器如QQ浏览器、360浏览器等等），Firefox
